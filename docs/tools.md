# Tools & Software

Everything AIRI uses, in one place. The **Classroom Track** needs only the first two
sections; the rest support the advanced Extension & Research stages.

!!! note "About third-party tools"
    AIRI is brand-agnostic — these are tools that have worked well, not requirements. Some
    are hosted by other labs and **may occasionally be offline**; items marked *(offline)*
    were unreachable at last check. Our own [Virtual Root](https://dr-richard-barker.github.io/virtual-root/)
    simulator is hosted alongside AIRI specifically so it stays available.

## 📲 Data collection & growing
| Tool | What it's for | Used in |
|---|---|---|
| **[AstroCalibration sticker / AstroBotany Spectrum](https://www.stickermule.com/drb2025)** | Known-size, known-colour marker so software can calibrate photos | Stage I onward |
| **[Epicollect5](https://five.epicollect.net/)** | Collect & share data from a phone, no account needed | Stages II–IV |
| **[Hamama kit](https://www.hamama.com/)** | Quick, mess-free microgreen growing | Stage III |
| **[Microgreen density calculator](https://jscalc.io/calc/vqHxgaGIt3RncroW)** | Work out ideal seeding density | Stage III |
| **Google Science Journal** | Use a phone's sensors to log light, sound, etc. | Stage III |

## 🔬 Image analysis
| Tool | What it's for | Used in |
|---|---|---|
| **[PlantCV](https://plantcv.org/)** | Plant computer vision: detect the sticker, colour-correct, measure plants & roots | Stages I, III, IV |
| **[ImageJ / Fiji](https://imagej.net/software/fiji/)** | General scientific image measurement | General |
| **[Easy Leaf Area](https://github.com/heaslon/Easy-Leaf-Area)** ([Android](https://play.google.com/store/apps/details?id=com.heaslon.EasyLeafArea)) | Measure leaf area from a photo | Stage III |
| **SmartRoot** | Semi-automated root tracing in ImageJ | Stages IV+ |
| **[RootNav 2.0 / RootNavPortal](https://github.com/Chagrilled/RootNavPortal)** | Root system architecture from images | Stages IV+ |
| **AstroDart** | Root growth & gravitropism analysis | Stages IV, VI |
| **[SOAPP](https://github.com/lvbauer/soapp-tutorial/blob/main/soapp-guide.md)** / **[web-glyphics](https://github.com/lvbauer/web-glyphics)** | Analysis using the AstroBotany Spectrum sticker | Stage I, III |

## 🧮 Modelling & simulation
| Tool | What it's for | Used in |
|---|---|---|
| **[Virtual Root](https://dr-richard-barker.github.io/virtual-root/)** ⭐ | Our open, interactive auxin-transport simulator (rebuilds SimuPlant) | Stage VII |
| SimuPlant | Original desktop auxin model *(offline — superseded by Virtual Root)* | Stage VII |

| **[shinyRootBox](https://plantmodelling.shinyapps.io/shinyRootBox/)**, **[archidart](https://plantmodelling.shinyapps.io/archidart/)**, **[PlaNet-Maize](https://plantmodelling.shinyapps.io/PlaNet_Maize/)** | Interactive root/plant modelling apps | Stages VIII–IX |
| **[RootFit](https://github.com/RootFitProject/rootfit)** | Root growth curve analysis | Stage III, IV |
| **[Photosynthesis model](https://biocycle.atmos.colostate.edu/shiny/photosynthesis/)** | Interactive photosynthesis simulation | Stage IX |
