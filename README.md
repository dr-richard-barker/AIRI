# AIRI: Astrobotany International Research Initiative

AIRI is an educational and research program focused on Astrobotany. This repository contains the source material for the AIRI documentation and curriculum.

## Documentation

The full documentation is available as a website:

[**AIRI Documentation Website**](https://dr-richard-barker.github.io/AIRI/)

### Running Locally

To run the documentation site locally for development:

1.  **Install dependencies:**
    ```bash
    pip install -r requirements.txt
    ```
2.  **Start the development server:**
    ```bash
    mkdocs serve
    ```
3.  **View the site:**
    Open your browser to `http://127.0.0.1:8000/`.
